How it works

  1. Pick a tracked job (auto-fills) or paste a posting.
  2. Build kit in Claude (MCP mode) opens a chat with live connector access: Claude fetches the live posting, researches the company and salary, and produces a tailored resume, cover letter, emails, and interview questions, then saves both a .md and a .docx file to your Application Kits folder and pushes the kit back to Job Squire.
  3. Build kit via API opens a live status page in a new tab (like the other automatic AI tasks) and calls whatever AI provider you've configured in Settings > AI (Claude, OpenAI, or a free-tier provider like Ollama, Groq, Gemini, or OpenRouter). It runs a best-effort web search first to research the company and salary range, then generates the kit, working from the profile, job description, and research findings. If the posting is saved as a job, the kit text is saved to the job record and a matching .docx file is attached to that job so you can download it from the job's detail page like any other file. If you didn't save it as a job, the kit is shown on the status page itself, with a button to download it as .docx before you navigate away.
  4. Downloading the kit file gives you the full prompt to paste into any AI assistant yourself, if you'd rather run it manually.
  5. On save, the kit is ATS-cleaned (em-dashes, curly quotes, and fancy bullets become plain ASCII) and the resume and cover letter are auto-saved as PDF attachments on the job.
